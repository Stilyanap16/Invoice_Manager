Описание:
--------------
GUI приложение за управление на транзакции.

Функционалности:
---------------
- Import от Excel
- Export към Excel
- CRUD операции
- Търсене и филтриране
- Откриване на общ приход, неплатени суми и дублирани фактури
- Графики

Инсталация:
----------
pip install -r requirements.txt

Стартиране:
----------
python app.py
